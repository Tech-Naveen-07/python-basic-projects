# 🔐 Python Password Generator

A command-line Password Generator built with Python that creates secure and customizable passwords based on user preferences.

## ✨ Features

- Generate passwords of custom length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special symbols
- Password strength analysis
- Save passwords to a text file
- Generate multiple passwords in one session

## 🛠️ Technologies Used

- Python
- random
- string
- File Handling

## 🚀 How to Run

```bash
python Password_generator.py
```
