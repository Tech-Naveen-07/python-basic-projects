# 📝 Python To-Do List

A simple **command-line To-Do List application** built with Python. It lets users manage tasks and stores them permanently using a text file.

## ✨ Features

- 👀 View tasks
- ➕ Add tasks
- ❌ Remove tasks
- 💾 Automatic file saving
- 📂 Persistent storage using `todo_list.txt`
- ✔️ Basic input validation

## 🛠️ Built With

- Python 3
- Lists
- Loops
- Conditionals
- File Handling
- String Methods

## ▶️ How to Run

```bash
git clone https://github.com/your-username/todo-list.git
cd todo-list
python todo.py
```
